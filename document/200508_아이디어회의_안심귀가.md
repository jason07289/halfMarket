# 안심귀가 (탈락 아이디어)

## 활용데이터

CCTV, 파출소위치, 가로등, 범죄내역, 생활안전지도(가능하면)

## 기능

### 메인 기능

1. 지도에서 저 데이터들 보기

2. 내 위치 공유 / 경로 공유 - 실시간 -> 불가능

### 부가 기능

추후 논의 예정

## 특징

웹앱

## 사용할 기술

    백 : Spring

    프론트 : Vue

    DB : MySQL vs MariaDB

## 논의 사항

- 웹사이트가 종ㄹ되어도 위치 추적이 계속 가능할것인지?

    -> 웹에 접속해있을때 정해진 시간마다 요청을 보내서

    -> 가능한지 여부를 알아봐야 할 듯
